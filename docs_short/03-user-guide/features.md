# Feature Walkthrough

## Feature 1: Guest Mode

**Overview:** Quick start without registration; progress not saved.

**How to Use:**

1. Launch the game
2. Select **Guest Mode**
3. Press **Start Game**

**Result:** First level loads immediately; progress and achievements are not saved.

**Tip:** Best for quick testing or casual play; registration unlocks saving and profile features.

---

## Feature 2: User Registration & Profile

**Overview:** Persistent progress, achievements, and user-specific info.

**How to Use:**

1. Select **Register / Login**
2. Complete registration or login
3. Open **User Profile**
4. Play the game

**Result:** User profile displays progress, statistics, and achievements; level completion updates progress and achievements.

---

## Feature 3: Autonomous Enemy Agents

**Overview:** Enemies use state-based behavior (idle, patrol, attack) and react to player and environment.

**How to Use:**

1. Enter a level
2. Interact/fight enemies using the **F** key
3. Observe type-specific behaviors

**Result:** Enemies switch states; player can be attacked or defeat enemies.

---

## Feature 4: Rope Interaction Mechanics

**Overview:** Ropes enable traversal, interaction, and puzzle-solving.

**How to Use:**

1. Enter a level
2. Find a rope (green for climbing, black for puzzle/button)
3. Interact according to tutorial hint

**Result:** Player progresses in the level, climbs ledges, and activates buttons as needed.

---

## Keyboard Shortcuts

| Shortcut      | Action                                   |
| ------------- | ---------------------------------------- |
| A/D           | Move Left/Right                          |
| W/S           | Look Up/Down                             |
| E             | Interact with objects                    |
| F             | Attack                                   |
| Space         | Jump / Double Jump                       |
| Q             | Ultimate Attack (charged by enemy kills) |
| Arrow Up/Down | Climb                                    |
| R             | Release rope                             |
