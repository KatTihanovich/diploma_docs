## Getting Started

### System Requirements

| Requirement | Minimum              | Recommended               |
| ----------- | -------------------- | ------------------------- |
| OS          | Windows 10, macOS 12 | Latest version            |
| Internet    | Optional             | Required for all features |
| Device      | Desktop              | -                         |

### Accessing the Application

1. Download Honey&Bunny.zip from Google Drive
2. Extract the archive
3. Open HoneyAndBunny.exe

### Step 1: Choose Game Mode

* **Guest Mode:** Play immediately; progress not saved
* **Register/Login:** Access profile, save progress, and achievements

### Step 2A: Guest Mode

* Taken directly to **Main Menu**
* Start a new game or select a level
* Progress and achievements **not saved**

### Step 2B: Registration/Login

* Register new account or login
* Complete form and wait for confirmation
* Redirected to **Main Menu**

### Step 3: Main Menu

* **Start Game** — begin gameplay
* **Select Level** — choose level
* **User Profile** *(registered users only)* — view progress and stats
* **Settings** — adjust volume and controls
* **Exit Game** — close application

### Step 4: Play

* Press **Start Game** to enter the level with tutorial hints and controls
